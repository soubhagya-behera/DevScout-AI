# DevScout AI

AI-powered GitHub Developer Analyzer for Recruiters.

## Features

- GitHub Profile Analysis
- Repository Analysis
- Technology Detection
- Developer Scoring Engine
- AI-Powered Insights (Gemini AI)
- Interview Question Generation (Coming Soon)

## Tech Stack

- Java 17
- Spring Boot
- GitHub REST API
- Gemini AI
- Maven
